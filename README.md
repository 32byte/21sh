**CHELLE** most commonly known as `42sh` project

*Objective* Recode a functionnal shell with essential features, described as follows:

* line edition
* lexing
* parsing
* pipe handling
* redirections
* ~~simple job control~~
* signal handling
* filename expansion
* brace expansion
* variable expansion
* some builtins (set, hash...)
* shell options
* history handling
* autocompletion
